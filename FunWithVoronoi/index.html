⯀ Instructions ⯀

  • VD = Voronoi Diagram  ⬥  Site = Vertex encompassing one VD cell  ⬥  Cell = Self-Evident!
  • [ Left Mouse Click ]   and not hovering over existing site ⮕ creates new site.
  • [ Left Mouse Press ]   and over existing site ⮕ selects the site for dragging.
  • [ Left Mouse Drag ]   and site selected ⮕ moves it via mouse (see release behavior effects!)
  • [ Left Mouse Release ]   and in bounds ⮕ de-selects selected site as to 'drop it into place'.
  • [ Left Mouse Release ]   and OUT of bounds ⮕ removes site from VD (note: CANNOT be undone!)
  • [ 'M' Key Press ] ⮕   does 1 round of 'Min. Bound. Rect. Smoothing' (makes cells more uniform).
  • [ 'R' Key Press ] ⮕   resets sites to new random positions, then creates new VD thereof.
  • [ 'G' Key Press ] ⮕   toggles basic display grid.
