Random Gaussian Testbed

Instructions (QAD):

  • hovering mouse over a distribution shows its current number of samples
  • pressing mouse over a distribution swaps between 'midpoint' xor 'endpoint' based alignment (i.e. 'human lips' xor bellcurve shape, resp.)
  • pressing 'a' or 'A' key toggles 'auto populate' mode, which generates 'numAutoPts' random gaussian samples for each distribution
  • pressing 'c' or 'C' key resets histogram bin values (i.e. set size) to zero for each distribution, and is intended for use with auto-populate mode (i.e. to visualize the emergence of a normal distribution via random gaussian samples as the set size increases)
